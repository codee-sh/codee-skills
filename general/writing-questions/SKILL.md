# Writing Questions12

Apply these rules every time you write a list of questions for the client.

## Rules

1. Use short hyphens "-" only. Never use long dashes "--" or em dashes "—".
2. Do not overuse hyphens. Use periods and commas where they are more natural.
3. Write in first person singular - "I", not "we".
4. Keep sentence structure simple and direct.

## Example

Bad:
> If a merchandiser adds or removes a group-by attribute — the entire fan-out must be rebuilt from scratch — should the system flag this for human review?

Good:
> If I add or remove a group-by attribute on an existing product, the entire fan-out needs to be rebuilt. Should the system flag this for human review before executing anything?
